# 📱 FlowOS – Product Requirements Document (PRD)

## 1. Overview
FlowOS is a high-performance **personal life operating system** designed to synchronize a user's biological state with their daily execution. It goes beyond simple task management by integrating habits, journaling, and proactive **Agentic Intelligence** to optimize consistency, focus, and reflection.

---

## 2. Strategic Goals
*   **Biological Synchronization**: Align daily actions with energy levels and circadian rhythms.
*   **Execution Mastery**: Provide a frictionless "Flow" for completing daily rituals and objectives.
*   **Agentic Intelligence**: Move from passive assistance to proactive "Neural Auditing."
*   **Absolute Data Sovereignty**: Ensure all user data remains local-first and encrypted.

---

## 3. Core Features

### 3.1 Neural Dashboard (Home)
*   **Neural Orb**: A central visualizer representing system sync through complex canvas-based animations.
*   **Flow Score (FS)**: A proprietary metric calculating execution efficiency across tasks and habits.
*   **Oracle HUD**: Proactive AI suggestions delivered through a high-fidelity banner interface.

### 3.2 Execution Flow (Tasks & Habits)
*   **Unified List**: A combined view of "Daily Rituals" (Habits) and "Objectives" (Tasks).
*   **Energy-Aware Scheduling**: Tasks tagged with energy requirements (1-10) for AI-driven momentum tracking.
*   **Glassmorphic UI**: Premium expanding cards with 1dp subtle borders and haptic feedback.

### 3.3 Reflection System (Journal / Sync)
*   **Vibe Tracking**: Daily 1-10 rating of cognitive/physical state.
*   **AI Insight Bridge**: Automated analysis of journal entries to identify recurring behavior patterns.
*   **Biometric Vault**: Journal history protected by a mandatory biometric (Fingerprint/Face) gateway.

### 3.4 Neural Interface (Jarvis AI)
*   **Omni-Input Command Center**: A voice-enabled, glassmorphic overlay for rapid intent execution.
*   **Neural System Audit**: Background workers that perform full system audits every 15-30 minutes to identify flow divergences.
*   **BYOK Protocol**: Multi-provider LLM support (OpenAI, Anthropic, Google, Nvidia, Groq) with model-specific tuning.
*   **Jarvis Waveform**: Custom animation visualizer for active voice listening.

---

## 4. System Architecture

### 4.1 Intelligence Layer
*   **SystemContext Model**: A structured view of the entire user state (Tasks, Habits, Vibes, Stats).
*   **Intent Parser**: A context-aware mapping engine that translates natural language into system commands.
*   **OracleWorker**: An autonomous agent that triggers proactive notifications based on real-time state analysis.

### 4.2 Security Layer
*   **Local-First Persistence**: Room DB for all user data.
*   **Encrypted API Keys**: System-level encryption for LLM keys.
*   **Identity Lock**: Biometric verification for sensitive protocol screens.

---

## 5. Design System: "Deep Obsidian"
*   **Theme**: Pure OLED Black (#0B0B0F).
*   **Accents**: Electric Blue (#00A3FF) and Arctic White.
*   **Aesthetics**: Glassmorphism, 24dp-32dp corner radii, and 1dp alpha-blended borders.
*   **Typography**: Modern Sans-Serif (Inter/Roboto) with extreme weight contrast.

---

## 6. Success Criteria
Users achieve "Level 5 Protocol Sync" by maintaining an average Flow Score of >75 and completing 90%+ of their Daily Rituals through proactive AI assistance.
